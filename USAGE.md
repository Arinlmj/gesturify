# GESTURIFY 使用手册

## 启动

1. 在项目目录运行 `npm install` 安装依赖。
2. 运行 `npm run dev -- --port 5173` 启动本地服务。
3. 打开终端显示的本地地址，例如 `http://127.0.0.1:5176/`。

## 摄像头手势模式

1. 点击「启动摄像头」。
2. 浏览器询问摄像头权限时选择允许。
3. 把手放入左侧摄像头区域，等待状态变成「追踪中」或提示「手部锁定」。
4. 拇指和食指靠近时进入「抓取」状态，右侧远处、中层、近处的粒子会分层向手势点收束成发光粒子核。
5. 保持抓取并左右或斜向拖动手势点，粒子球、旋涡环或几何场会跟随拖动旋转；不拖动时，粒子不会自动自转。
6. 完全松开拇指和食指并保持短暂打开后进入「释放」状态，聚合粒子会从释放位置以球形向外绽放。

## 控制项

- 「能量」控制粒子的飞散速度和爆发力度。
- 「粒子密度」控制一次释放产生的粒子数量。
- 「旋转灵敏度」控制抓取后手势点拖动到粒子旋转的速度映射；设为 0 时抓取也不会旋转。
- 「特效样式」可切换「粒子球」「旋涡环」「几何场」三种抓取聚拢效果；释放时统一为粒子球外射。
- 摄像头旁的暂停按钮会停止摄像头。
- 网格按钮会显示或隐藏左侧扫描网格。

## 状态指标

- 「锁定度」表示当前手势识别置信度。
- 「捏合距离」表示拇指和食指之间的检测距离。
- 「当前动作」显示当前处于「抓取」或「释放」。
